PROMETHEUS

Confidential physics-as-a-service with honest simulation provenance.

Problem

In science platforms, the request can leak the research program. A shared simulator should not automatically learn which scenario a lab is running.

Product

Type a physics task. Get a routed simulation, frame-by-frame output, dataset provenance, model status, and privacy-preserving retrieval.
